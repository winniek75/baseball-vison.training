# WISE Baseball Coaching Analyzer v3.1

WISE Baseball Academy 内部コーチング動画分析ツール

## 機能

### 🎬 分析画面
- スロー再生（1/10〜等速）／コマ送り（±1フレーム）／左右反転
- 描画ツール：矢印・直線・楕円・四角・フリーハンド・角度計測・テキスト（7色・太さ3段階・アンドゥ/リドゥ）
- フレーム＋描画をPNG書き出し／レポートへ添付（📎）
- 音声解説つき分析動画の録画 → ブラウザ内MP4変換（iPhoneのLINEで再生可）
- 録画中に描いた線はリアルタイムで映像に反映

### 🆚 比較画面（2画面）
- **各画面が独立**：個別に再生/停止・速度（1/10〜等速）・シークバー・±1コマ・左右反転
- **各画面に直接描き込み**：矢印・丸・直線・フリーハンド（画面ごとにクリア/戻す）
- 同時操作も可能：同時再生/停止・同時±1コマ・同期速度
- 動画ソース：ファイル／分析中の動画／解説録画／ドリルのお手本

### ✍ 返信作成
- 良い点（プリセット＋自由入力）・改善点・選択ドリルから、LINEにそのまま貼れる下書きを組み立て → ワンタップコピー
- 指導履歴：生徒名・タグで検索して再利用
- **📄 レポート出力**：注釈入りキャプチャ＋文面を、印刷/PDF保存できる整形レポートに（従来手作業のPDFをデジタル化）

### 📚 ドリルライブラリ
- タグ付きストック、チェック選択でメニュー作成
- お手本動画をアップロードして使い回し（ブラウザ内IndexedDB保存）／分析画面・比較画面で再生可能

## 使い方

```bash
npm install
npm run dev
```

## デプロイ（Vercel）

GitHubにプッシュ後、Vercelが自動デプロイします。

## Tech Stack

- React 18 / Vite 5
- HTML5 Canvas / Video API
- @ffmpeg/ffmpeg（wasm・MP4変換）

## 注意

- ドリル・履歴はブラウザのlocalStorage、お手本動画はIndexedDBに保存されます（端末内）。PC変更時はエクスポート機能でテキストデータをバックアップ、動画の元ファイルは別途保管を。

## v3.1 追加機能
- 🎙 **比較画面ごと解説録画**：2画面＋描き込みを音声つきで録画 → MP4変換（LINE送付可）
- 📸 **比較キャプチャ**：2画面＋描き込みを1枚画像にしてレポートへ
- 🗂 **生徒カルテ**：生徒名を入力すると過去の返信（最新3件）を返信作成画面に表示
- 💬 **定型フレーズ管理**：よく使う指導の言い回しを登録し、タップで改善点へ挿入
- 🎞 **連続コマ写真**：現在位置からNコマを横並び1枚に自動合成（スイング連続写真）
- 🖼 **LINE用画像出力**：文面＋画像を縦長1枚PNGに合成してそのまま送付
